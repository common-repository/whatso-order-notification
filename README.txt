== Description ==

**Instantly owner and customer get order notifications!**
Every customer loves to receive acknowledgment for successful orders as it gives them surety that the order is placed. And, our plugin does exactly the same by sending an automatic message to customers.

As a site owner, you have put a lot of hard work to set up your Business, Technology, and Marketing and are waiting for the orders. Our plugin understands your excitement and immediately Messages you for every successful order.

Feel the rush on every successful order!

== Installation ==

= From Dashboard ( WordPress admin ) =
* plugins -> Add New
* search for 'Order Notification by Whatso'
* click on Install Now and then Active.

= using FTP or similar =
* Unzip "Whatso" file and
* Upload "Whatso" folder to the "/wp-content/plugins/" directory.
* Activate the plugin through the "Plugins" menu in WordPress.
